📊 Sales Dashboard Analysis
📌 Project Overview

This project presents an interactive sales dashboard created using Excel to analyze and visualize order data. The dashboard helps in understanding business performance by providing insights into sales trends, customer behavior, order status, and regional performance.

It is useful for data analysis beginners, business analysts, and decision-makers to quickly interpret large datasets through visual representation.

🎯 Objectives

Analyze overall sales performance

Identify top-performing categories and products

Understand customer demographics (Age, Gender)

Track order status and sales channels

Visualize regional sales distribution

Support data-driven business decisions

📂 Dataset Information

The dataset contains order-level transactional data including:

Order ID

Customer ID

Gender

Age

Order Date

Order Status

Sales Channel

Product Category

Quantity

Sales Amount

Shipping City & State

Currency

✨ Interactive Filters / Slicers
